Using the Timer

You can set the timers by clicking on the appropriate button and using the up and down arrows.


Delay Timer: Time for you to prepare. This is how much time will pass before the bell rings.


Meditation Timer: The amount of time you want to meditate.


Interval Timer: Set this timer if you want a single bell to ring in the middle of your meditation. If not, set this to zero.


Choosing a Bell

There are three bells to choose from. Click on each bell button to preview. The bell you choose will chime three times at the start and end of your meditation.
